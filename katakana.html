As you will have noticed, the カタカナ and ひらがな both share the same assoisiated sounds.
This is because all ひらがな words can be spealt with the カタカナ, and all カタカナ words can be spelt with the ひらがな.
Below you will find the direct translations:

ア = a, イ = i, ウ = u, エ = e, オ = o
カ = ka, キ = ki, ク = ku, ケ = ke, コ = ko
サ =sa, シ = shi, ス = su, セ = se, ソ = so
タ = ta, チ = chi, ツ = tsu, テ = te, ト = to
ナ = na, ニ = ni, ヌ = nu, ネ = ne, ノ = no
ハ = ha, ヒ = hi, フ = fu, ヘ = he, ホ = ho
マ = ma, ミ = mi, ム = mu, メ = me, モ = mo
ラ = ra, リ = ri, ル = ru, レ = re, ロ = ro
ヤ = ya, ユ = yu, ヨ = yo
ワ = wa, ヲ = wo
ン = n

NB: It's important to know that the カタカナ is used for spelling foreign words in Japanese, a vast majority of writen Japanese will appear as ひらがな.

Diacritics

The カタカナ Diacritics are pretty much exactly the same as the ひらがな, so no explaination is required.

ザ = za, ジ = ji, ズ = zu, ゼ = ze, ゾ = zo

パ = pa, ピ = pi, プ = pu, ペ = pe, ポ = po

Digraphs

They're the same as the ひらがな Digraphs, so expect nothing new.

Congratulations! You've just learned the 'Kana', which means that you can read 2/3 of a Japanese sentence; although the 漢字 still stands in your way and it only gets harder from here.
